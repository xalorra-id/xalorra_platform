## ğŸŒ API Prediksi

Endpoint HTTP untuk melakukan prediksi:

```http
POST /predict/<model_name>
```

### Contoh:

```bash

  -H "Content-Type: application/json" \
  -d '{"features": [9, 1]}'
```

```json

  "model": "xgb-v2",
  "input": [9, 1],
  "prediction": 10
```
